Holy Paladin (хилер) реализован ~80%.

## Реализовано
- **Поиск самого дохлого** в группе/рейде (GetNumRaidMembers → raid1..N, party1..4)
- **Beacon of Light** (Частица Света) на фокус-таргет — автоподдержание
- **Sacred Shield** (Священный щит) на фокус — автоподдержание
- **Divine Plea** (Святая клятва) при мане < 70% — только с бурстом (Гнев карателя / Божественное просветление / тринкеты)
- **Хилы**: Шок небес (инстант), Свет небес (основной), Вспышка Света (OFF по умолч.)
- **Экстренные**: Божественное одобрение (< 30%), Возложение рук (< 15%)
- **Порог хила**: bestHP >= 1.0 → не хилим (держит всех на фул)
- **Правосудие**: приоритет 0 — поддерживать бафф "Безупречное правосудие" (только в бою); низкий приоритет — дебафф на таргете если все здоровы

## UI (радио-выборы в баффах)
- **Выбор печати**: Печать мудрости (SoW) / Печать Света (SoL)
- **Выбор правосудия**: Правосудие мудрости (JoW) / Правосудие света (JoL) — ВНИМАНИЕ: "света" с маленькой буквы!
- **Выбор благословения**: Могущества / Королей / Мудрости (+ Великое если есть реагент "Знак королей")
- **Выбор ауры**: 6 аур (воздаяния, воина Света, лёд/огонь/тьма, сосредоточенности) — общая для всех палов

## Ключевые нюансы
- Правосудие кастуется ТОЛЬКО в бою (UnitAffectingCombat) — чтобы не пулить мобов
- "Правосудие света" — маленькая "с"! WoW spell names case-sensitive
- Буффы (печать/благо/аура) проверяют КОНКРЕТНЫЙ баф, а не "любую печать" — при смене сразу перебафает
- IsHealer=true → PreChecks пропускает combat/target проверки
- BuffScript работает даже без обычных тоглов баффов (печать/благо/аура достаточно)

**Why:** Нужно знать текущее состояние хпала для дальнейшей доработки
**How to apply:** При работе с хпалом — проверять эти приоритеты и имена спеллов

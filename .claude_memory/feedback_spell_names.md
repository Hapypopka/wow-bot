WoW спеллы на WoWCircle (русский клиент) — case-sensitive! Правила:

1. **Никогда не использовать букву "ё"** — в клиенте все "е" (Звездный, а не Звёздный)
2. **Регистр важен**: "Правосудие света" (маленькая с), "Печать Света" (большая С)
3. **Перед добавлением нового спелла** — сверять с скриншотом спеллбука

Известные ошибки регистра:
- "Правосудие Света" ✗ → "Правосудие света" ✓
- "Символ королей" ✗ → "Знак королей" ✓
- "Скверноброня" ✗ → "Доспех Скверны" ✓
- "Похищение души" ✗ → "Ожог души" ✓
- "Опустошение" ✗ → "Истребление" ✓

**Why:** Неправильное имя = CastSpellByName молча не кастует, отладка занимает много времени
**How to apply:** При добавлении любого нового спелла — сверять с скриншотом или GetSpellInfo(id)

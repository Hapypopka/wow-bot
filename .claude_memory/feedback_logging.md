Логировать всё в файл wowbot.log через Logger.cs, не через DEFAULT_CHAT_FRAME в WoW.
**Why:** На этапе разработки нужна полная картина для дебага. Брат на другом ПК — может просто скинуть лог-файл.
**How to apply:** Использовать Logger.Info/Warn/Error для любой диагностики. Чат WoW не засорять дебаг-сообщениями.

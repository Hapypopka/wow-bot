Русские названия спеллов WoW 3.3.5a брать с сайта: `https://wow.66wan.net/80/ru/?spells=CLASS.TAB.CATEGORY`

Пример: `https://wow.66wan.net/80/ru/?spells=7.11.574` — спеллы друида.

**Why:** Названия на сайте совпадают с русским клиентом WoWCircle. Раньше приходилось гадать с переводом и регистром.
**How to apply:** При добавлении новой ротации — сначала открыть нужный класс на этом сайте, собрать spell ID + русские имена, потом писать код.

# Правила для автофикса через Telegram бот (@clwowbot)

1. МИНИМАЛЬНЫЕ ИЗМЕНЕНИЯ — фикси ТОЛЬКО баг, не рефактори
2. НЕ создавай новые файлы — правь существующие
3. Проверяй компиляцию: `dotnet build WowBot.sln`
4. Класс-изоляция — палладинские фичи ТОЛЬКО для PALADIN, курсы ТОЛЬКО для WARLOCK
5. Тестировщики — не разработчики, отвечай просто
6. Логи — если скинул wowbot.log, ОБЯЗАТЕЛЬНО прочитай перед диагностикой
